Never Lose a Word

  • Auto-Save Every keystroke is saved. No manual save, no lost work. Resume exactly where you left off.
  • Visual Diff Compare any two versions with a visual diff. See additions in green, deletions in red, moves highlighted.
  • Branching Create a branch to experiment with a rewrite. Keep the original safe. Merge when you are happy with the result.
  • Named Snapshots Save named snapshots at important milestones. "Draft 1", "Editor Review", "Final". Jump to any snapshot instantly.
  • Restore Restore any previous version with one click. Or cherry-pick specific paragraphs from older versions.
Collaboration

Write Together, No Conflicts

Real-time collaborative editing powered by CRDTs. Multiple writers, one document, zero conflicts. Every character converges to the same state, guaranteed.

01

Real-Time Editing

See every collaborator's cursor and text as they type. Sub-100ms latency. No turn-taking, no locks. Everyone writes simultaneously.

02

Conflict-Free (CRDT)

Built on Conflict-free Replicated Data Types. Even if two people edit the same sentence at the same time, the result is always consistent. Mathematically guaranteed.

03

Comments & Review

Inline comments anchored to specific text. Threaded discussions. Suggested edits that the author can accept or reject. Review workflow built in.
